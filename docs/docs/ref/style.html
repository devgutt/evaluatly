Below are the parameters to customize the style of the story.

All parameters are optional.

# Text

font_size

Reference size of the text.

text_font_color

Text color.

font_family

Font family. It can be a system font or a Google font.

align

Alignment of items.

Values: left (default), center, or right.

title_font_color

Title reference font color.

question_font_color

Question reference font color.

# Background

background_color

Page background color.

background_img_src

URL of an image to be used as background.

Multiple images and some effects, like gradient, can be used.

This is equivalent to the CSS property background-image.

background_img_size

Size of the background image.

Values: cover (default), auto, or contain.

This is equivalent to the CSS property background-size.

background_img_repeat

The pattern of repetition of the background image.

Values: no-repeat (default), repeat, repeat-x, repeat-y, space, or round.

This is equivalent to the CSS property background-repeat.

background_img_position

Position of the background image.

Values: center (default), top, bottom, left, right.

The values can be combined and edge offset added (e.g. bottom 10px right 20px)

This is equivalent to the CSS property background-position.

background_color_content

Content background color. Defaults to transparent.

# Button

button_bg_color

Button reference background color.

button_font_color

Button reference font color.

button_border_radius

Button reference border-radius.

button_border_width

Button reference border width for buttons.

# Forms

input_back_color

Form input reference background color.

input_font_color

Form input reference font color.

input_accent_color

Form input reference accent color.

input_error_color

Form input error color.

# Examples

{
    "font_size": "20px",
    "text_font_color": "#6c6c6c",
    "font_family": {"type" : "google", "name" : "Lato"},
    "align": "center",
    "title_font_color" : "#000000",
    "question_font_color" : "#000000",
    "background_color": "#f5f1ed",
    "button_bg_color": "#f1574b",
    "button_font_color": "#ffffff",
    "button_border_radius" : "100px",
    "button_border_width" : "3px",
    "input_accent_color" : "#3e306e"
}

Example using a system font:

{
    "font_family": "Arial, Roboto"
}

Example with repeating background image:

{
    "background_img_src": "url(https://domain.com/image.png)",
    "background_img_size": "auto",
    "background_img_repeat": "repeat",
    "background_img_position": "0%"
}
